# Sentence unscrambler

Available at https://imax531.github.io/unscrable-sentence/

Have a scrambled mess you want to arrange into a coherent sentence?
Just input the letters and sentence configuration to solve the scramble!

## Example

The the scrabled letters ```NEHSCOIAEPOWIHAFIF``` and sentence configuration ```6 2 4 6``` we get the sentence ```Coffee shop in hawaii```
